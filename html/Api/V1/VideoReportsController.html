    GET /video_reports

    Retrieves all script reports based on Api::V1::ErrorReportsController#get_reports method and returns them as a JSON response. Each with the following fields:

    • ‘id`: integer.

    • ‘reportType`: string.

    • ‘description`: string.

    • ‘status`: string.

    • ‘priority`: integer.

    • ‘author`: string.

    • ‘eMail`: string.

    • ‘videoTitle`: string.

    • ‘timestampStart`: string.

    • ‘timestampEnd`: string.

    • ‘videoURL`: string.

    • ‘report_date`: string in the format YYYY-MM-DD.

    • ‘granted_date`: string in the format YYYY-MM-DD.

    • ‘completed_date`: string in the format YYYY-MM-DD.

    • ‘university_module_id`: integer.

    • ‘mitify_user`: hash.

      • ‘name`: string.

      • ‘first_name`: string.

      • ‘email`: string.

      • ‘role`: hash.

        • ‘title`: string.
